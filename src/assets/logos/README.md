# Institution Logos

## Required Logo Files

Please add the following logo files to this directory:

### 1. BIT College Logo
- **Filename:** `bit-logo.png`
- **Description:** Official Bangalore Institute of Technology logo (green shield with coat of arms)
- **Recommended size:** 512x512 pixels or higher
- **Format:** PNG with transparent background preferred

### 2. Department Logo (DevSoc)
- **Filename:** `devsoc-logo.png`
- **Description:** ISE Department Developer's Society logo (graduation cap icon)
- **Recommended size:** 512x512 pixels or higher
- **Format:** PNG with transparent background preferred

## How to Add Logos

1. Save the BIT college logo as `bit-logo.png` in this folder
2. Save the DevSoc logo as `devsoc-logo.png` in this folder
3. The application will automatically detect and display them

## Fallback

If logo images are not found, the application will fall back to text-based logos (BIT and ISE circles).
